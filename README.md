# Career Landing

Одностраничный сайт для онлайн-обучения профессиям. Проект подготовлен для деплоя на Vercel через GitHub.

## Как запустить локально

```bash
npm install
npm run dev
```

Откройте http://localhost:3000

## Как загрузить на GitHub

1. Создайте новый репозиторий на GitHub.
2. Загрузите все файлы из этой папки в репозиторий.
3. Откройте Vercel → Add New Project → Import Git Repository.
4. Выберите репозиторий.
5. Framework Preset: Next.js.
6. Нажмите Deploy.

## Что заменить перед публикацией

- Telegram: `@education_mentor`
